# 02 - Java Basics

## Objetivo

En este módulo aprenderé los conceptos fundamentales de Java como lenguaje de programación.

Al finalizar este módulo seré capaz de:

- Comprender cómo funciona Java internamente.
- Entender la diferencia entre JDK, JRE y JVM.
- Conocer el proceso de compilación y ejecución.
- Crear clases y objetos.
- Definir métodos y constructores.
- Utilizar miembros estáticos (`static`).
- Organizar proyectos mediante paquetes (`packages`).
