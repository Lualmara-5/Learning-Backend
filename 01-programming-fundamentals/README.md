# Programming Fundamentals

Esta sección contiene los conceptos básicos de programación que sirven como base para aprender cualquier lenguaje.

Antes de aprender Java, Python, C++ o cualquier otra tecnología, es importante comprender cómo piensa un programa y cómo se manipulan los datos.

## Objetivos

Al finalizar esta sección debería ser capaz de:

- Comprender qué es una variable.
- Tomar decisiones mediante condiciones.
- Repetir tareas utilizando ciclos.
- Almacenar múltiples datos en arreglos.
- Resolver problemas simples mediante programación.

## Temas

### 1. Variables

Aprender cómo almacenar información dentro de un programa.

### 2. Conditions

Aprender cómo tomar decisiones dependiendo de una condición.

### 3. Loops

Aprender cómo repetir instrucciones sin escribirlas varias veces.

### 4. Arrays

Aprender cómo almacenar múltiples valores en una misma estructura.
